Etapes du projet

  1. Initialiser le projet
  2. Ajouter React Router
  3. Créer le routeur et les différentes pages

  5. Gérer la redirection vers la page d'erreur
  6. Vérifier l'ensemble du site

Compétences évaluées

  • Initialiser une application avec Create React App
  • Configurer la navigation avec React Router
  • Développer les éléments de l'interface grâces aux composants React
  • Développer une interface web avec Sass
  • Mettre en oeuvre des animations CSS

Description

Pour ce cinquième projet, j'ai d'abord initialisé l'application React avec create-react-app puis j'ai configuré le router avec react-router-dom à l'aide du gestionnaire de paquets npm. J'ai mis en place les différentes routes vers les pages d'accueil (Home), d'hébergements (Hosting), d'à propos (About) et de gestion des erreurs (Error). Pour construire mes pages, j'ai d'abord créé des composants React en utilisant des props auxquels j'ai ensuite passé les valeurs appropriées provenant de mon fichier JSON avec la fonction map si nécessaire. Pour mettre en place les animations, j'ai notamment utilisé le hook d'état UseState afin de configurer ma flèche rotative en mode ouverte/fermée (Toggle) ou encore les flèches de mon carrousel. Pour la mise en forme, j'ai utilisé Sass pour structurer mon CSS et j'ai également rendu le contenu adaptable aux différents supports (Design responsive). En cas d'erreur sur l'identifiant d'un hébergement dans l'URL, j'ai géré les redirections avec le composant Navigate et le hook UseNavigate.
