Description

Pour ce premier projet, découper préalablement la maquette Figma (mobile < 768px, tablette >= 768px, desktop > 1024px) m'a permis de mieux structurer ma page HTML. J'ai utilisé les flexbox pour organiser les éléments que ce soit verticalement ou horizontalement. Pour les cartes d'hébergement, je me suis inspiré des snippets de code de W3C et j'ai adapté les images avec la propriété CSS "Object-fit". Les Média Queries ont été utilisées pour rendre la page adaptable aux diverses tailles d'écran, optimisant ainsi l'expérience utilisateur. La mise en page a été réalisée à l'aide des Chrome DevTools permettant d'ajuster la largeur de l'écran et d'observer les modifications nécessaires. La balise méta viewport a été intégrée pour optimiser le rendu, la lisibilité et l'adaptabilité de la page aux différents supports. Afin de rendre le code plus clair, j'ai utilisé la règle @import afin de trier soigneusement les CSS concernant le header, le main, le footer ainsi que les Média Queries. Enfin, j'ai vérifié la conformité de mon code aux standards du web à l'aide des validateurs HTML et CSS du W3C.

Difficulté(s) rencontrée(s)

  • Prise en main des Médias Queries
